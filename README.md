# ice-box

Laptop stand air-cooling control system

## Milestones

M1 - switch control
M2 - fixed fan power
M3 - fan PWM fixed/cycle
M4 - fan tach serial
M5 - fan PWM potentiometer control
M6 - temperature serial
M7 - temperature OLED
M8 - control temp threshold
M9 - control temp PID
M10 - LEDs display temp, tach on OLED, more TBD
